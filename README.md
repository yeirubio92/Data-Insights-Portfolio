# Data-Insights-Portfolio
Portafolio de análisis de datos y reportes estadísticos aplicados al sector agroindustrial (Aguacate Hass y Mercados).
# Data Insights Portfolio - Agro & Business Intelligence

## 🎯 Objetivo
Este repositorio compila mis proyectos finales de análisis de datos, donde transformo datos crudos en insights accionables para la toma de decisiones estratégicas en el sector agropecuario. 

## 📂 Proyectos Destacados

### 🥑 1. Análisis de Rendimiento y Costos (Aguacate Hass)
- **Descripción:** Estudio de la correlación entre fertilización, clima y costo por kilo producido.
- **Herramientas:** Python (Pandas, Matplotlib) / R (ggplot2).
- **Entregable:** Reporte ejecutivo con recomendaciones de ahorro.

### 📈 2. Dinámicas de Precios en Mercados Mayoristas
- **Descripción:** Visualización de tendencias de precios históricos y predicción de ventanas de mercado.
- **Herramientas:** RStudio (Tidyverse) y series de tiempo.
- **Entregable:** Dashboard interactivo.

## 🛠️ Skills Técnicas
- **Lenguajes:** R, Python, SQL.
- **Análisis:** Estadística descriptiva, Limpieza de datos (ETL), Visualización de datos.
- **Soft Skills:** Comunicación de datos técnicos para perfiles no técnicos (gerencia).
